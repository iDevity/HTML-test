What is Meshcaid?

Meshcaid is intended to be a Free-to-Play Sandbox where anything can be created where the only limitation is ones imagination. Users can build their own 3D Models. Aswell, as code their own scripts to create their own game. Users will be able to create their own clothing designs aswell as outfit their characters with many different appearal options.
